设置Java程序网络代理

February 17, 2021 · 1 min read

Alan

配置来源

To configure a Java application to send web traffic to Fiddler, set the proxy using jre:

jre -DproxySet=true -Dhttp.proxyHost=127.0.0.1 -Dhttp.proxyPort=8888
