Vim 相关配置及技巧

基本配置#

Windows配置文件为 ~/_vimrc, Unix配置文件为 ~/.vimrc

配置内容为:

filetype plugin on
set rnu
set nu
set nocompatible
set noswapfile
set noswf
colo koehler
set encoding=utf-8
set fileencoding=utf-8
set autoindent
set showcmd
set shiftwidth=4
set tabstop=4
set softtabstop=4
